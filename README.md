# 금융 감성 분석

## 실험 목적
국내에서 보도되는 주식 관련 기사들이 실제 주식의 가격 또는 움직임에 영향을 주는지 알아본다.

## 실험 가정
* 이 프로젝트에서는 전날의 감성 점수가 다음날 주가에 반영될 것이라 가정하고 실험을 진행하였다.
* 이 프로젝트에서는 삼성전자의 감성 점수 및 주가를 사용하였다.

## 실험 방법
### 1. 감성점수로 예측한 주가의 상승 및 하락
* 감성 점수가 0보다 클 때 상승, 0보다 작을 때 하락, 0일 때 유지로 예측한다.
* 이를 실제 주가와 비교하여 그 정확도를 산출한다.
### 2. 감성점수를 독립 변수로 가지는 다중선형회귀 모형 분석
* 감성 점수와 코스피200 전체 산업군 지수를 독립 변수로 갖는 다중회귀모형을 수립한다.
  1. Y: 삼성전자 주가, X: KOSPI200 전체 산업군 지수, 삼성전자 감성 점수
  2. Y: 코스피 지수, X: KOSPI200 전체 산업군 지수, 삼성전자 감성 점수
* Backward Elimination을 진행해 최적의 모형을 도출한다.

## 데이터 수집
* 삼성전자 관련 뉴스 데이터는 네이버 증권으로부터 Selenium 패키지를 사용해 삼성전자 관련 기사들의 헤드라인을 크롤링하여 수집하였다.
* 삼성전자 주가와 코스피 지수 데이터는 야후 파이낸스로부터 pandas-datareader 패키지를 사용해 수집하였다.
* 코스피200 전체 산업군 지수 데이터는 한국거래소(KRX)로부터 Pykrx 패키지를 사용해 수집하였다.

## 알고리즘 과정
1. k개의 클러스터 중심과 그에 따른 배정 정보 행렬을 초기화한다. 초기 클러스터 중심은 샘플로부터 k개의 클러스터 중심을 임의로 비복원 추출하여 설정한다.
2. 샘플을 각 클러스터 중심과의 거리를 계산해 가장 가까운 클러스터로 배정한다. 그에 따라 배정 정보 행렬을 업데이트한다.
3. 모든 샘플의 클러스터 배정을 완료하였다면, 각 클러스터에 속하는 샘플들의 평균을 새로운 클러스터 중심으로 업데이트한다.
4. 배정 정보 행렬이 바뀌지 않을 때까지 2번~3번의 과정을 반복한다.
5. 루프가 종료되었다면, 배정 정보에 따라 각 샘플을 클러스터 집합에 추가한다. 

## 실험 결과 분석
* 알고리즘을 여러 데이터 셋에 적용해본 결과, 대체로 원하는 좋은 결과를 얻을 수 있었다. 하지만 시도에 따라 결과가 달라지거나 무한 루프가 발생하여 결과를 도출하지 못하는 경우가 발생했다. 분포가 밀집된 데이터 셋에서 자주 무한 루프가 발생하였으며, 이상치 데이터를 클러스터에 포함하면서 오차가 발생하는 경우 또한 종종 있었다.
* 이러한 문제의 원인을 분석한 결과, 알고리즘의 성능이 초기 클러스터 중심에 의해 크게 좌우됨을 확인할 수 있었다.

## 개선방안
* K-Menas++: 샘플 중 1개를 임의로 추출해 그로부터 거리가 가장 먼 샘플들로 초기 클러스터 중심을 설정하는 방법, 초기 클러스터 중심을 전략적으로 배치하는 방법으로 기존의 방법보다 더 나은 결과와 수렴 속도를 기대할 수 있다.
* 루프 종료 조건을 거리에 대한 임계값으로 설정: 무한 루프를 방지할 수 있다.
