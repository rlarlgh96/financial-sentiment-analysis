# 금융 감성 분석

## 실험 목적
국내에서 보도되는 주식 관련 뉴스들이 실제 주식의 가격 또는 움직임에 영향을 주는지 알아본다.

## 실험 가정
* 이 프로젝트에서는 전날 감성 점수가 다음날 주가에 반영될 것이라 가정하고 실험을 진행하였다.
* 이 프로젝트에서는 삼성전자의 감성 점수 및 주가를 사용하였다.

## 실험 방법
### 1. 감성점수로 예측한 전날 대비 주가의 움직임
* 전날 감성 점수가 0보다 클 때 상승, 0보다 작을 때 하락, 0일 때 유지로 예측한다.
* 이를 실제 주가와 비교하여 그 정확도를 산출한다.
### 2. 다중선형회귀 모형 분석
* 감성 점수와 코스피200 전체 산업군 지수를 독립 변수로 갖는 다중회귀모형을 수립한다.
  1. Y: 삼성전자 주가, X: KOSPI200 전체 산업군 지수, 삼성전자 감성 점수
  2. Y: 코스피 지수, X: KOSPI200 전체 산업군 지수, 삼성전자 감성 점수
* 감성 점수의 독립 변수가 제거되기 전까지 Backward Elimination을 진행해 최적의 모형을 도출한다.

## 데이터 수집
* 뉴스 데이터는 네이버 증권으로부터 Selenium 패키지를 사용해 삼성전자 관련 기사들의 헤드라인을 크롤링하여 수집하였다.
* 삼성전자 주가와 코스피 지수 데이터는 야후 파이낸스로부터 pandas-datareader 패키지를 사용해 수집하였다.
* 코스피200 전체 산업군 지수 데이터는 한국거래소(KRX)로부터 Pykrx 패키지를 사용해 수집하였다.
* 2022년 1월 1일부터 2022년 11월 30일까지의 데이터를 수집하였다.

## 데이터 전처리
* 감성 점수 산출에는 VADER 패키지를 사용하였다.
* VADER 패키지는 영문만 지원하고 있어 수집한 뉴스 데이터를 파파고를 통해 영문으로 변역하여 저장하였다.
* VADER 패키지로 산출한 네 종류의 점수 중 종합 점수를 감성 점수로 사용하였다.
* 각 뉴스 데이터의 감성 점수를 일자별로 평균하여 합산하였다.
* 앞선 실험의 가정에 따라 데이터 인덱스를 활용해 테이블 간 날짜를 맞추는 작업을 진행하였다. 이와 동시에 감성 점수 테이블에서 주식 휴장일에 해당하는 데이터를 제거하는 작업을 진행하였다.

## 실험 결과
### 1. 감성점수로 예측한 주가의 상승 및 하락
* 앞선 실험 방법에 따라 주가의 움직임을 상승은 1, 하락은 -1, 유지는 0으로 예측하였다.
* 그 결과 42.53%의 정확도를 보였다.
### 2. 다중선형회귀 모형 분석
* 앞선 실험 방법에 따라 감성 점수의 독립 변수가 제거되기 전까지 Backward Elimination을 진행하였다.
* 도출한 최적의 모형은 다음과 같다.
1. Y: 삼성전자 주가, X: KOSPI200 전체 산업군 지수, 삼성전자 감성 점수
